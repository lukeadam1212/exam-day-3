vsc code is used to compile the code and run local servers

1_frontend

installing react:
npx create-react-app

installing packages:
npm i react-router-dom,
npm i styled-components,
npm i axios

launching react app:
npm start

2_backend

installing node:
npm init

installing packages:
npm i cors,
npm i express,
npm i node-fetch,
npm i mongoose,
npm i nodemon,
npm i colors

launching node.js
npm run nodemon
